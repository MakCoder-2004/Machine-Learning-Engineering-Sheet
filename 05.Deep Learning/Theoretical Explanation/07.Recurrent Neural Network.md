# Recurrent Neural Networks (RNNs)

Recurrent Neural Networks (RNNs) are a class of neural networks designed to handle **sequential data** such as time series, text, or speech. Unlike feedforward networks, RNNs have **cyclical connections**, allowing them to maintain a hidden state that captures information about previous elements in the sequence.

---

## 1. Basic RNN Architecture

A standard RNN processes a sequence of inputs $(x_1, x_2, ..., x_T)$ and produces outputs $(y_1, y_2, ..., y_T)$ using a hidden state $h_t$ updated at each timestep.

### **Equations:**

$$
h_t = \tanh(W_{xh} x_t + W_{hh} h_{t-1} + b_h)
$$

$$
y_t = W_{hy} h_t + b_y
$$

Where:

* $x_t$ : input at time $t$
* $h_t$ : hidden state at time $t$
* $y_t$ : output at time $t$
* $W_{xh}, W_{hh}, W_{hy}$ : weight matrices
* $b_h, b_y$ : biases

---

## 2. Types of RNNs

RNNs can be classified based on input-output configurations:

| Type             | Description                                                    |
| ---------------- | -------------------------------------------------------------- |
| **One-to-One**   | Standard feedforward network                                   |
| **One-to-Many**  | Single input, sequence output (e.g., image captioning)         |
| **Many-to-One**  | Sequence input, single output (e.g., sentiment analysis)       |
| **Many-to-Many** | Sequence input and sequence output (e.g., machine translation) |

---

## 3. Vanishing and Exploding Gradients

During backpropagation through time (BPTT), gradients are propagated backward through multiple timesteps. This can cause:

* **Vanishing Gradients:** Gradients shrink exponentially, making long-term dependencies hard to learn.
* **Exploding Gradients:** Gradients grow exponentially, leading to unstable training.

### **Mathematical Intuition:**

$$
\frac{\partial L}{\partial h_t} = \sum_{k=t}^{T} \frac{\partial L}{\partial h_k} \prod_{j=t+1}^{k} W_{hh}^T , \text{diag}(\tanh'(h_j))
$$

* If $|W_{hh}| < 1$, gradients **vanish**.
* If $|W_{hh}| > 1$, gradients **explode**.

### **Solutions:**

* Gradient clipping
* LSTM / GRU architectures
* Proper weight initialization

## 4. Long Short-Term Memory (LSTM)

* LSTM is an RNN variant designed to **capture long-term dependencies** and mitigate vanishing gradients.
* 3 Gates (Input, Output and Forget)

### **Core Components:**

1. **Forget Gate** ($f_t$): decides what to discard from the cell state.

$$
f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)
$$

2. **Input Gate** ($i_t$) & Candidate Memory ($\tilde{C}_t$): decides what new information to add.

$$
i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)
$$

$$
\tilde{C}*t = \tanh(W_C [h*{t-1}, x_t] + b_C)
$$

3. **Cell State Update** ($C_t$):

$$
C_t = f_t * C_{t-1} + i_t * \tilde{C}_t
$$

* Long Term Memory

4. **Output Gate** ($o_t$) & Hidden State ($h_t$):

$$
o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)
$$

$$
h_t = o_t * \tanh(C_t)
$$
* Short Term Memory <br>
* Where $\sigma$ is the sigmoid function and $*$ denotes element-wise multiplication.

### **Advantages:**

* Retains long-term dependencies
* Mitigates vanishing gradient problem
* Widely used in NLP, speech, and time-series tasks

---

## 5. Gated Recurrent Unit (GRU)

* GRU is a simplified alternative to LSTM that **combines forget and input gates** into a single update gate.
* 2 Gates (reset, update)

### **Equations:**

1. **Update Gate** ($z_t$) and **Reset Gate** ($r_t$):

$$
z_t = \sigma(W_z x_t + U_z h_{t-1} + b_z)
$$

$$
r_t = \sigma(W_r x_t + U_r h_{t-1} + b_r)
$$

2. **Candidate Activation** ($\tilde{h}_t$): (Long Term Memory)

$$
\tilde{h}*t = \tanh(W_h x_t + U_h (r_t * h*{t-1}) + b_h)
$$


3. **Hidden State Update:** 

$$
h_t = (1 - z_t) * h_{t-1} + z_t * \tilde{h}_t
$$
 
### **Advantages:**

* Fewer parameters than LSTM
* Comparable performance
* Faster to train

---

## 6. Bidirectional RNN (BiRNN)

Bidirectional RNNs process sequences in **both forward and backward directions** to capture past and future context.

### **Equations:**

Forward pass:

$$
\overrightarrow{h_t} = \tanh(W_{xh} x_t + W_{hh} \overrightarrow{h_{t-1}} + b_h)
$$

Backward pass:

$$
\overleftarrow{h_t} = \tanh(W_{xh} x_t + W_{hh} \overleftarrow{h_{t+1}} + b_h)
$$

Final output:

$$
y_t = W_y [\overrightarrow{h_t}; \overleftarrow{h_t}] + b_y
$$

### **Applications:**

* Machine translation
* Speech recognition
* Named entity recognition

| Concept                   | Purpose                                                       |
| ------------------------- | ------------------------------------------------------------- |
| RNN                       | Sequence modeling; hidden states capture temporal information |
| LSTM                      | Handle long-term dependencies; mitigate vanishing gradients   |
| GRU                       | Simplified LSTM with fewer parameters                         |
