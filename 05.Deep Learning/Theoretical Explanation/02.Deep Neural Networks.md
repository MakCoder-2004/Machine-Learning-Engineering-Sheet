# Deep Neural Networks

A **Deep Neural Network (DNN)** is a stacked composition of many (L ≥ 2) hidden layers, each made of the same atomic unit:
**affine transform + element-wise non-linearity**.<br>
Going deep lets the model build **hierarchical representations** (edges → motifs → objects → meaning) and crack problems that stay out of reach for shallow nets.

---

## The Neuron

| Biological term  | Deep-Learning analogue | Math meaning                          |
| ---------------- | ---------------------- | ------------------------------------- |
| Dendrites        | Input vector **x**     | Raw features                          |
| Synaptic weights | Weight matrix **W**    | Learnable importance per feature      |
| Cell body        | Pre-activation **z**   | $z = W x + b$                         |
| Nucleus          | Bias vector **b**      | Layer-wise offset                     |
| Axon hillock     | Activation **g(z)**    | Non-linear “fire or not”              |
| Axon terminal    | Activation **a**       | Signal sent **forward** to next layer |

> One layer = **#neurons** such units in parallel.

---

## Anatomy of a Deep Network
```
Input  →  Hidden 1  →  Hidden 2  → … →  Hidden L  →  Output
a⁰  →    a¹      →    a²      → … →    aᴸ      →  aᴸ⁺¹
```

* **a⁰** = raw features
* **aˡ** = $g^{(l)}( W^{(l)} a^{(l-1)} + b^{(l)} )$ for $l = 1 … L+1$
* Depth = L hidden layers (everything before the output).

---

## Getting Your Matrix Dimensions Right

| Tensor             | Shape                  | How to remember                      |
| ------------------ | ---------------------- | ------------------------------------ |
| Mini-batch X       | $(n_x, m)$             | “features × examples”                |
| $W^{(l)}$          | $(n^{(l)}, n^{(l-1)})$ | “current neurons × incoming neurons” |
| $b^{(l)}$          | $(n^{(l)}, 1)$         | “one bias per neuron”                |
| $Z^{(l)}, A^{(l)}$ | $(n^{(l)}, m)$         | “outputs × examples”                 |

> Pro-tip: draw the **$(n^{(l)}, m)$** rectangle once; every line must match.

---

## Forward Propagation in a Deep Network

For layer $l = 1 … L+1$:

1. Linear step:
   $Z^{(l)} = W^{(l)} A^{(l-1)} + b^{(l)}$
2. Non-linear step:
   $A^{(l)} = g^{(l)}(Z^{(l)})$

Vectorised across **m** examples in one shot.

---

## Building Blocks of Deep Neural Networks

| Block      | Role                                                                    |
| ---------- | ----------------------------------------------------------------------- |
| Linear     | $Z = W A + b$                                                           |
| Activation | $A = g(Z)$                                                              |
| Loss       | $J = f(A^{(L+1)}, Y)$                                                   |
| Backward   | $dW, db = \frac{\partial J}{\partial W}, \frac{\partial J}{\partial b}$ |
| Optimiser  | $W := W - \alpha dW$                                                    |

Stack these five primitives L times and you have a **Deep L-layer Network**.

---

## Why Deep Representations?

* **Level 1** (edges, phonemes)
* **Level 2** (corners, syllables)
* **Level 3** (textures, words)
* **Level 4** (objects, phrases)
* **Level 5** (faces, sentiment)

Each layer **re-uses** previous detectors → **exponentially** more expressive with only linear extra parameters.

## Activation Functions Cheat-Sheet

| Function   | Equation                          | Range        | Hidden?   | Output?       |
| ---------- | --------------------------------- | ------------ | --------- | ------------- |
| ReLU       | $\max(0,z)$                       | [0,∞)        | ✅ default | ❌             |
| Leaky-ReLU | $\max(0.01z,z)$                   | (-∞,∞)       | ✅         | ❌             |
| ELU        | $z$ if $z>0$ else $\alpha(e^z−1)$ | (-α,∞)       | ✅         | ❌             |
| Swish      | $z \sigma(\beta z)$               | ≈(-0.7,∞)    | ✅ SOTA    | ❌             |
| GELU       | $z \Phi(z)$                       | ≈(-0.17,∞)   | ✅ NLP     | ❌             |
| Sigmoid    | $\frac{1}{1+e^{-z}}$              | (0,1)        | Legacy    | ✅ binary      |
| Tanh       | $\frac{e^z−e^{-z}}{e^z+e^{-z}}$   | (-1,1)       | Shallow   | ❌             |
| Softmax    | $\frac{e^{z_i}}{\sum_j e^{z_j}}$  | (0,1) sums 1 | ❌         | ✅ multi-class |
| Linear     | $z$                               | (-∞,∞)       | ❌         | ✅ regression  |

## Forward & Backward Propagation

### Forward<br>
$A^{(0)} = X$<br>
for $l = 1 … L+1$:<br>
 $Z^{(l)} = W^{(l)} A^{(l-1)} + b^{(l)}$<br>
 $A^{(l)} = g^{(l)}(Z^{(l)})$<br>

### Loss
$J = \frac{1}{m} \sum Loss(a^{(L+1)}, y)$<br>

### Backward
$dA^{(L+1)} = \frac{\partial J}{\partial A^{(L+1)}}$<br>
for $l = L+1 … 1$:<br>
 $dZ^{(l)} = dA^{(l)} \odot g'^{(l)}(Z^{(l)})$<br>
 $dW^{(l)} = \frac{1}{m} dZ^{(l)} A^{(l-1)T}$<br>
 $db^{(l)} = \frac{1}{m} \sum dZ^{(l)}$<br>
 $dA^{(l-1)} = W^{(l)T} dZ^{(l)}$<br>

### Update
$W^{(l)} := W^{(l)} - \alpha dW^{(l)}$<br>
$b^{(l)} := b^{(l)} - \alpha db^{(l)}$<br>

## Parameters vs Hyperparameters

| Parameters                  | Hyperparameters                                                                                                                                                           |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $W^{(l)}, b^{(l)}$ (learnt) | #layers L, #units $n^{(l)}$, learning-rate $\alpha$, activation $g^{(l)}$, batch-size, epochs, dropout p, weight-decay $\lambda$, optimiser (Adam, RMSprop, SGD-momentum) |

> Grid-search, random-search, or **Bayesian optimisation** for the right column.

## What Does This Have to Do With the Brain?

* **Biological neurons** spike; artificial ones **float**.
* **Synapses** store strength; **weights** store gradients.
* **Cortex is deep**; so are our networks.

Still, **back-prop** is not biologically plausible yet the **hierarchical representation** idea is borrowed wholesale.

## Training Tips for Very Deep Nets

| Trick                      | Benefit                          |
| -------------------------- | -------------------------------- |
| **Xavier / He** init       | Keeps variances stable           |
| **Batch-Norm**             | Reduces internal covariate shift |
| **Skip / Residual**        | $F(x)+x$ trains 1000+ layers     |
| **Learning-rate schedule** | Cosine, warm restarts            |
| **Data augmentation**      | More data for free               |
| **Dropout**                | Ensemble in a single net         |
| **Early-stopping**         | Stop when val-loss ↑             |

---

## When to Go Deep

✅ Millions of examples <br>
✅ Hierarchical structure (images, text, speech)<br>
✅ Simple models under-fit <br>

❌ Tabular data with 10 k rows → **Gradient Boosting** usually wins.<br>
❌ Need full interpretability → stay shallow.<br>

---

## One-Sentence Summary

> Stack **L affine + non-linear** blocks, feed them **data + gradients**, and they will **automatically learn** a hierarchy of features that crushes any shallow learner on complex tasks.
