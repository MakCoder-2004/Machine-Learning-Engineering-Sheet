## Features & Capabilities

spaCy supports:

* Tokenization
* Lemmatization
* POS tagging
* Dependency parsing
* Named Entity Recognition
* Sentence segmentation
* Word vectors
* Rule-based matching
* Text categorization
* Custom pipeline components
* Efficient batch processing with `nlp.pipe()`

## Matcher vs PhraseMatcher vs EntityRuler Comparison

| Feature           | Matcher              | PhraseMatcher      | EntityRuler    |
| ----------------- | -------------------- | ------------------ | -------------- |
| Pattern type      | Token patterns       | Phrases            | Token + phrase |
| Speed             | Fast                 | Very fast          | Medium         |
| Ideal for         | Fine-grained control | Multi-word phrases | Custom NER     |
| Creates entities? | No                   | No                 | **Yes**        |
| Regex support     | ❌                    | ❌                  | ❌              |

# Applying Concepts by Code

## Load spaCy

```python
import spacy
nlp = spacy.load("en_core_web_sm")
text = "Apple is looking at buying U.K. startup for $1 billion."
doc = nlp(text)
```

## Rule-Based Matcher

```python
from spacy.matcher import Matcher
matcher = Matcher(nlp.vocab)
pattern = [{"LOWER": "apple"}]
matcher.add("APPLE_PATTERN", [pattern])
matches = matcher(doc)
print(matches)
```

Advanced example:

```python
pattern = [
    {"POS": "PROPN"},
    {"LOWER": "is"},
    {"POS": "VERB"}
]
matcher.add("PROPER_IS_VERB", [pattern])
for match_id, start, end in matcher(doc):
    print(doc[start:end].text)
```

---

## Phrase Matcher

```python
from spacy.matcher import PhraseMatcher
matcher = PhraseMatcher(nlp.vocab)
phrases = ["New York", "Los Angeles", "San Francisco"]
patterns = [nlp(text) for text in phrases]
matcher.add("CITIES", patterns)
for match_id, start, end in matcher(doc):
    print(doc[start:end].text)
```

More examples:

```python
brands = ["iPhone 15", "Samsung Galaxy", "PlayStation 5"]
matcher = PhraseMatcher(nlp.vocab)
matcher.add("BRANDS", [nlp(b) for b in brands])
d = nlp("I bought an iPhone 15 and a PlayStation 5 yesterday.")
for _, s, e in matcher(d):
    print("Matched:", d[s:e].text)
```

## Text Classification (Pipeline Component)

```python
import spacy
from spacy.util import minibatch
from spacy.training.example import Example

nlp = spacy.blank("en")
tc = nlp.add_pipe("textcat")
tc.add_label("POSITIVE")
tc.add_label("NEGATIVE")

train_data = [
    ("I love this product!", {"cats": {"POSITIVE": 1, "NEGATIVE": 0}}),
    ("This is terrible.",    {"cats": {"POSITIVE": 0, "NEGATIVE": 1}}),
]

optimizer = nlp.initialize()
for i in range(10):
    losses = {}
    batches = minibatch(train_data, size=2)
    for text, annotations in batches:
        doc = nlp.make_doc(text)
        example = Example.from_dict(doc, annotations)
        nlp.update([example], losses=losses)
    print("Losses:", losses)

print(nlp("I really enjoy this!").cats)
```

## Custom Pipeline Components

### Simple Component

```python
def hello_component(doc):
    print("Pipeline running on:", doc.text)
    return doc

nlp.add_pipe(hello_component, first=True)
_ = nlp("Hello world!")
```

### Component Adding Custom Attributes

```python
from spacy.tokens import Doc
Doc.set_extension("word_count", default=0)

def word_count_component(doc):
    doc._.word_count = len([t for t in doc if t.is_alpha])
    return doc

nlp.add_pipe(word_count_component, last=True)
output = nlp("Apple is buying a startup.")
print(output._.word_count)
```

## Language Processing on a Pre-trained Pipeline

**What this does:** 
- This allows you to inspect the components of a spaCy model and see what tasks it performs automatically (tokenization, tagging, parsing, NER, etc.). You can access, analyze, or extend each component.


```python
nlp = spacy.load("en_core_web_sm")
print(nlp.pipeline)
```

Output example:

```python
[('tok2vec', <spacy.pipeline.tok2vec.Tok2Vec object at 0x...>),
 ('tagger', <spacy.pipeline.tagger.Tagger object at 0x...>),
 ('parser', <spacy.pipeline.dep_parser.DependencyParser object at 0x...>),
 ('ner', <spacy.pipeline.ner.EntityRecognizer object at 0x...>),
 ('attribute_ruler', <spacy.pipeline.attributeruler.AttributeRuler object at 0x...>),
 ('lemmatizer', <spacy.pipeline.lemmatizer.Lemmatizer object at 0x...>)]
```
You can run text through this pipeline and each component will automatically annotate the Doc object.
